## Installation

Run the firebase login command

`firebase login`

## Usage

Pushing to git will not automatically update the host.
When you are ready to deploy code to the host, first make sure of the following:
1) The latest version is pulled down from github
2) `npm run build` was successful
3) All new changes are committed, pushed and the git working tree is clean

Then run the firebase deployment command

`firebase deploy`
